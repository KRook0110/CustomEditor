# Shaditor

## About
This is a project for practicing my C and memory management.

This project is not finished.

## Dependencies
1. gcc

## How to use
1. Make the program
`make`
2. Run the editor
`./shaditor {file_name}`
3. Move around with the arrow keys
4. Save the file press {ctrl + s}
5. Quit from the editor press {ctrl + q}
